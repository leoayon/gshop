**谷粒外卖移动端应用** | (前端开发) 

- **项目描述**: 一款仿主流外卖平台的移动端Web应用，旨在提供流畅的线上点餐体验。
- **工作职责**:
  - **负责**商品列表、购物车、用户中心等核心模块的前端开发工作。
  - **遵循**模块化、组件化的开发思想，封装了多个可复用的UI组件。
- **技术实现与亮点**:
  - 项目基于 **`Vue 2`** + **`Vue Router`** + **`Vuex`** 全家桶进行开发，使用 `Webpack` 构建。
  - **实现**了购物车功能，通过 `Vuex` 集中管理购物车状态，解决了商品列表、商家详情、购物车页面之间的**跨组件状态同步**问题。
  - **封装**了如 `Star`（星级评分）、`CartControl`（加减按钮）等高复用性业务组件，提高了开发效率。
  - 应用了图片**懒加载**（vue-lazyload）技术，优化了商品大图的加载
- **技术栈**: `Vue 2`, `Vuex`, `Vue Router`, `Webpack`, `ES6`, `Stylus`
